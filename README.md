# emotion-auto-css README

if you want it to work in jsx、tsx and ts. You need add this setting in vscode:

```
"editor.quickSuggestions": {
    "strings": true
}
```

You can get the style name prompt in normal format in this code.

```
<div css={css``}></div>
```
